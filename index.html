Wie erstelle ich einen Token? (Anleitung anzeigen)
  1. Klick auf den Link oben — du landest direkt auf der GitHub Token-Seite
  2. Bei „Note" einen Namen eingeben, z.B. „VANtastic Kanban"
  3. Bei „Expiration" wählen wie lange der Token gilt (z.B. 90 Tage)
  4. Bei „Select scopes" muss repo angehakt sein (sollte schon vorausgewählt sein)
  5. Ganz unten „Generate token" klicken
  6. Den angezeigten Token (ghp_...) kopieren und hier einfügen
  7. Wichtig: Der Token wird nur einmal angezeigt — bei Verlust einen neuen erstellen
